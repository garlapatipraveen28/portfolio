Modern Personal Portfolio Website built with HTML, CSS, and JavaScript.

Overview

Praveen Portfolio is a responsive and interactive personal portfolio website designed to showcase professional experience, technical skills, featured projects, certifications, and contact information through a modern UI.

The project focuses on responsive frontend development, reusable UI sections, smooth animations, accessibility, and professional web presentation practices.
